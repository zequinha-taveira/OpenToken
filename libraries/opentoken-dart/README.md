# OpenToken Dart SDK

The official **Shared Logic** core for the OpenToken ecosystem.

## Usage
Add this to your Flutter `pubspec.yaml`:
```yaml
dependencies:
  opentoken_dart:
    path: ../libraries/opentoken-dart
```

## Features
- **Shared OATH Logic**: APDU generation and response parsing.
- **Platform Agnostic**: Works on Android, iOS, Windows, Linux, and macOS.
- **Protocol Fidelity**: Implements ISO7816 OATH standards.
