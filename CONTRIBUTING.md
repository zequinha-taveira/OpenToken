# Contributing to OpenToken

First off, thank you for considering contributing to OpenToken! It's people like you that make OpenToken such a great tool.

## 100% Open Source & Independence (NATIVO)
OpenToken is committed to being 100% Open Source and independent of proprietary brands. We follow a "Zero Yubico" policy for our native ecosystem.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues before opening a new one.
* Use a clear and descriptive title.
* Describe the exact steps which reproduce the problem.

### Pull Requests
* Focus on standard protocols (FIDO2, OATH, OpenPGP).
* Maintain the security-first architecture.
* Update documentation for any new features.

## Security Policy
Please do not report security vulnerabilities through public GitHub issues. Email the maintainers directly or use the responsible disclosure program.
